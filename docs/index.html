Instalar a App pelo Site

No telemovel, abre esta pagina e carrega em Descarregar APK. Depois abre o ficheiro descarregado e confirma a instalacao.

1. Abrir no telemovel Abre este site no Chrome ou noutro browser Android.
2. Descarregar Carrega no botao Descarregar APK.
3. Instalar Abre o ficheiro em Downloads e aceita a instalacao.
4. Atualizar Quando houver versao nova, descarrega outra vez por este mesmo site.
