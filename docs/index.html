Instruções para o Assistente

Mesmo quando estás fora de casa no telemóvel, podes enviar novos pedidos e tarefas para o assistente AI (Claude) no teu PC.

1. Ficheiro no Google Drive 📁
Cria um ficheiro de texto chamado instrucoes.txt dentro da pasta TripGuard no teu Google Drive do telemóvel. O assistente lê este ficheiro automaticamente a cada sincronização!
2. Criar uma Tarefa no GitHub 💬
Cria um "Issue" ou tarefa no repositório GitHub para o assistente implementar na próxima execução.
